## 手机在做什么

手机是在现有局域网中运行代理服务器。它不会创建 Wi-Fi 热点，也不会替代家里的路由器。

```text
                代理请求                              普通出站连接

电脑/电视/游戏机  <---->  Wi-Fi 路由器  <---->  Android 手机  <---->  互联网
                                                   HTTP :8080
                                                   SOCKS5 :1080
```

客户端连接手机的局域网 IP。手机再通过 Android 当前默认网络建立目标连接。如果手机同时开启了另一款 VPN，该 VPN 通常会成为代理服务器的出站线路。

## 选择协议

| 协议 | 默认端口 | 适用情况 |
| --- | ---: | --- |
| HTTP | `8080` | 客户端提供普通 HTTP 代理设置，或需要通过 `CONNECT` 访问 HTTPS |
| SOCKS5 | `1080` | 客户端明确支持 SOCKS5，并希望通过 `socks5h` 代理 DNS |

不要把 HTTP 端口配置成 SOCKS5，也不要把 SOCKS5 端口配置成 HTTP。两个服务都不需要用户名和密码。

## 配置步骤

1. 让 Android 手机和客户端设备连接同一个可信 Wi-Fi 或局域网。
2. 打开 Android Proxy Server，记下界面显示的局域网地址，例如 `192.168.1.2`。
3. 启用 HTTP、SOCKS5，或者同时启用两者。
4. 在客户端应用中填写手机地址和对应协议的端口。
5. 使用代理期间保持 Android 前台服务通知处于运行状态。

```text
HTTP 代理
主机：192.168.1.2
端口：8080
身份验证：无

SOCKS5 代理
主机：192.168.1.2
端口：1080
身份验证：无
```

请把 `192.168.1.2` 替换成应用实际显示的地址。

## 命令行测试

在 Windows PowerShell 中使用 `curl.exe`，避免调用 PowerShell 的 `curl` 别名。

```powershell
curl.exe --proxy http://192.168.1.2:8080 https://example.com
curl.exe --proxy socks5h://192.168.1.2:1080 https://example.com
```

流量路径如下：

```text
请求
客户端 -> 路由器 -> Android 代理 -> 当前 VPN 或默认网络 -> 目标网站

响应
目标网站 -> 当前 VPN 或默认网络 -> Android 代理 -> 路由器 -> 客户端
```

## 故障排查

### 无法连接端口

```text
客户端能否访问手机？
        |
        +-- 不能 -> 检查是否同一 Wi-Fi、访客网络、AP 隔离和防火墙。
        |
        +-- 能   -> 检查服务开关，并确认端口与协议一致。
```

部分路由器会禁止 Wi-Fi 客户端之间互相通信。必要时关闭访客网络隔离、AP 隔离或客户端隔离。

### HTTP 可以使用，但 SOCKS5 不行

- 确认客户端选择的是 SOCKS5，而不是 SOCKS4 或 HTTP。
- 如果没有在应用里修改端口，应使用 `1080`。
- 删除客户端中的用户名和密码；服务器不使用身份验证。
- 命令行需要让 DNS 也经过代理时，应使用 `socks5h://`，而不是 `socks5://`。

### 可以建立连接，但网页打不开

- 检查手机本身能否访问目标网站。
- 检查手机当前 VPN 或 Android 默认网络是否正常。
- 切换 VPN 状态后，停止并重新启动对应代理服务。

### Active 在 0 和少量数字之间变化

应用、浏览器和代理客户端可能会建立持续时间很短的健康检查或 DNS 相关连接。即使没有明显打开网页，Active 也可能快速变化。Session Total 会累计本次服务运行期间已经结束的连接尝试，并在服务停止后重置。

## 安全提示

代理会监听本机所有网络接口，并且没有身份验证。请只在可信局域网中使用，不需要时及时关闭，也不要在公网路由器上把 `8080` 或 `1080` 端口转发到手机。

更多信息参见[安全说明](../SECURITY.zh-CN.md)和[隐私说明](../PRIVACY.zh-CN.md)。
