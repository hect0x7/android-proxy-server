# 安全说明

[English](SECURITY.md) | 简体中文

## 支持版本

安全修复只应用于 `master` 分支上的最新版本。发布新版本后，不再维护更早的 APK 构建产物。

## 部署边界

Android Proxy Server 面向可信局域网使用。HTTP 和 SOCKS5 服务不提供身份验证，启用后会监听本机所有网络接口。

- 不要把代理端口直接暴露到公网。
- 不要在不可信的公共 Wi-Fi 或热点网络中启用代理服务器。
- 如果局域网内的客户端不可信，请使用 Android 防火墙或网络隔离措施。
- 不需要代理时，应关闭 HTTP 和 SOCKS5 服务。

## VPN 与回环防护

本应用不使用 `VpnService`。出站连接遵循 Android 默认路由，包括另一款已连接的 VPN。应用会拒绝来自手机自身的连接，以降低递归代理回环风险；外部代理链仍需要谨慎配置。

## 发布完整性

Release APK 由 GitHub Actions 构建。工作流会在上传产物前验证 APK 签名，并比较 APK 证书 SHA-256 与配置的发布 keystore 是否一致。签名私钥和密码不会存储在仓库中。

## 报告安全漏洞

不要在公开 Issue 中发布利用细节、凭据、私有网络地址或捕获的代理流量。优先使用仓库提供的私有漏洞报告通道；如果没有私有通道，只创建一个不含敏感信息的公开 Issue，请求建立私下联系。

报告应包含受影响版本、Android 版本、复现条件、预期行为和安全影响。提交前请删除 Token、密码、流量内容和个人信息。
